# Hello World Excel+Node.js

### setup
* using yarn add:
    * nodemon -D
    * express
    * read-excel-file
* using npm install:
    * [xlsx](https://www.npmjs.com/package/xlsx)
    
# How to run the code
   * run `yarn start` on terminal
   * goto localhot:3333
